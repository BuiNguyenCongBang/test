# DEFEND GAME
## Giới thiệu

Đây là một trò chơi kinh điển, nơi mục tiêu của người chơi là thủ nhà. Nhiệm vụ của bạn là tiêu diệt các quái vật.

## Cách chơi

- Dùng chuột để bắn.
- Dùng ```Space``` để nâng cấp đạn khi đầy mp
